This directory contains the Delphi/Pascal source for 
MakeGmap.exe
SendMessage.exe
SplitGmapSupp.exe

Other required sources can be found in the CommonUnits directory. This directory (and subs) is in the search path. '..\CommonUnits' etc.

The compiler used was Delphi 12.1 Community Edition. https://www.embarcadero.com/products/delphi/starter

Open Delphi\MkgMapTools.groupproj in Delphi and Build all Projects. Executables should be in the bin directory
